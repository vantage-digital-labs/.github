# Vantage Digital Labs

AI tooling for game teams. We build NPC dialogue workflows and prototyping tools for game developers — making AI-powered NPCs accessible to indie teams, not just enterprises.

**Current stage:** Early access. We're a small engineering team actively developing the NPC dialogue engine and iterating based on feedback from early users.

### What We Build

- **Cognitive NPC Engine** — Dynamic, in-character dialogue with emotional state tracking and multi-language support (EN, JA, KO, ZH)
- **Real-time Inference** — Low-latency NPC responses via WebSocket streaming
- **Game Engine SDKs** — Native integrations for Unity and Unreal Engine
